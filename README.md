# V_Camp-ToDo
Aplicativo To-Do criado como desafio da primeira semana do V_Camp :)
### Desktop JS

- [x] As tarefas devem ser mostradas na tela por ordem alfabética.
- [x] Os textos devem ter apenas a primeira letra maiúscula.
- [x] Ao clicar em uma tarefa que não esteja completa, ela deve ser completada.
- [x] Ao clicar em uma tarefa completada, ela deve ser transformada em não completada

### Desktop Style
- [ ] Em telas superiores a 1280px, o conteúdo deve se adaptar ao centro da tela.
- [x] As tarefas devem estar distribuídas em duas colunas
- [x] O conteúdo deve ter no máximo 1280px
- [ ] Tarefas não completadas devem conter um círculo sem fundo antes do texto
- [ ] Tarefas completadas devem conter um círculo com fundo antes do texto

### Mobile JS
- [x] As tarefas devem ser mostradas por ordem alfabética
- [x] Os textos devem ter apenas a primeira letra maiúscula.
- [x] Ao “tocar” em uma tarefa que não esteja completa, ela deve ser completada.
- [x] Ao “tocar” em uma tarefa completada, ela deve ser transformada em “não completada”
- [ ] Ao clicar no botão de adicionar tarefa, apresentar um modal que adiciona tarefa
- [ ] O Modal deve conter um campo para digitar a nova tarefa, um ícone para registrar a nova tarefa
- [ ] Ao clicar no botão de adicionar tarefa com o modal ativo, ele deve fechar o modal.
### Tablet
- [x] Criar botão de adicionar tarefa
- [ ] Telas de até 1024px.
- [ ] Esta versão deve ser criada pelo participante
